# Focus Timer Pro | 作業効率を最大化するポモドーロタイマー

シンプルかつ高機能な、ブラウザで動くポモドーロタイマーです。
「集中」と「休憩」のサイクルを視覚的にサポートし、あなたの作業や勉強の効率を最大化します。

##  サイトURL
[https://focus-timer-pro.com](https://focus-timer-pro.com)

##  特徴
- **フルスクリーン対応**: 没入感のある表示で、余計な情報を遮断して作業に没頭できます。
- **勉強・仕事に最適**: ポモドーロ・テクニック（25分集中＋5分休憩）に基づいた設計。
- **美しいアニメーション**: 集中状態を邪魔しない、洗練されたダークモードとUIデザイン。
- **カスタマイズ可能なテーマ**: 気分に合わせて配色を変更可能。
- **履歴管理機能**: 毎日の作業ログを自動保存。振り返りによるモチベーション維持。
- **完全無料**: 全ての機能を無料でご利用いただけます。

##  使い方
1. 設定パネルから、作業時間やテーマ、アラーム音を選択します。
2. 画面をタップ/クリックしてタイマーをスタート。
3. タイマー終了後、休憩を挟んでサイクルを繰り返します。
4. 履歴（ログ）画面で、これまでの頑張りを確認できます。
